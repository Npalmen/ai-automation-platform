# Product

## Product name
AI Automation Platform

## One-sentence description
En multi-tenant AI-driven operationsplattform som tar emot inkommande arbete, klassificerar det, extraherar nyckeldata, tillämpar policy, pausar för approval vid behov och exekverar åtgärder via integrationer.

## Target users
- Svenska små och medelstora bolag
- Service- och installationsbolag
- Kontorstunga verksamheter
- Senare större multi-entity / enterprise-kunder

## Core problem
Företag lägger oproportionerligt mycket tid på manuell hantering av e-post, leads, fakturor, kundärenden, approvals och uppföljning mellan system.

## Core value proposition
Plattformen minskar administrativ friktion genom att kombinera:
- strukturerad AI-bearbetning
- deterministiska workflows
- policybaserad automation
- mänsklig kontroll när risk eller osäkerhet kräver det

## Product goals
- Minska manuell handläggningstid
- Öka spårbarhet och kontroll
- Göra automation möjlig utan att ge fri AI kontroll över systemflödet
- Fungera som generell plattform, inte bara en enskild point solution

## Non-goals
- Inte bygga full enterprise suite i MVP
- Inte stödja alla integrationsfamiljer från start
- Inte optimera för perfekt UX före fungerande kärnflöde
- Inte bygga mobilapp i första fasen
